## Описание задачи

В рамках работы построен конвейер потоковой обработки данных на основе Apache Kafka для прогнозирования цены биткоина в реальном времени.  
Система имитирует сбор данных из CSV-файла, обрабатывает их, применяет модель машинного обучения и визуализирует результаты на дашборде.

**Ключевые компоненты:**

- train_model.py - обучение модели (сохраняется в папку models)
            RandomForestRegressor (100 деревьев, глубина 10)
            19 признаков
            R^2 = 0.9993 на тесте
- producer.py - читает исторические данные и отправляет их в kafka
- data_processor_consumer.py - вычисляет признаки (скользящее среднееб RSI, волатильность)
- ml_consumer.py - загружает предварительно обученную модель и делает предсказания
- visualization_consumer.py - отображает графики цены и предскаазаний в реальном времени
- dashboard.py


## Архитектура системы
Используются 2 брокера Kafka (вторая схема) с репликацией.
**Особенности:**
- 2 брокера (повышенная отказоустойчивость).
- Репликация фактора 2 и `min.insync.replicas=2` – гарантирует сохранность данных.
- Несколько Consumer'ов (обработка, ML, визуализация) работают параллельно.
- Producer отправляет данные с задержкой (`--delay`) для имитации реального времени.
